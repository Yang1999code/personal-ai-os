# 可复制到 ~/.codex/AGENTS.md 的精简规则

默认用中文回答；英文专有名词后加中文说明，例如 skill（技能）、loop（循环）、memory（记忆）、wiki（维基）。

我的个人 AI 操作系统：

- `~/Desktop/myskill` 是 skill（技能）入口层。
- `~/Desktop/myloop` 是 loop（循环）工作流层。
- `~/Desktop/my知识库` 是 Markdown（文档）/ wiki（维基）经验沉淀层。

工作规则：

- 简单问题直接回答。
- 复杂探索、长期任务、需要复盘的任务，优先考虑是否使用 `myloop`。
- 有复用价值的经验，优先沉淀到 `my知识库`。
- 遇到 `myloop`、`myskill`、`my知识库` 相关任务时，先读对应本地规则文件，不要凭印象回答。
- 不要默认修改我的文件；除非我明确说要写入、沉淀、更新或修改。
- 每次回答最后加“白话总结”。

长期目标：

帮我成为用好 AI 提效的“超级个体”。日常任务要尽量形成：使用 AI -> 产生经验 -> 沉淀 wiki -> 提炼 skill / loop -> 下次更快更稳。
