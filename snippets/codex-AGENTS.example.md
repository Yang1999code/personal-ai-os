# AGENTS.md 示例

## 默认沟通

- 默认用中文。
- 英文专有名词后加中文解释。
- 复杂概念用白话、类比、例子、Mermaid（流程图）。
- 每次回答最后加白话总结。

## 本地 AI 操作系统

- `myskill/` 是 skill（技能）入口层。
- `myloop/` 是 loop（循环）工作流层。
- `my知识库/` 是经验沉淀层。

## 边界

- 不要写入真实 token（令牌）、key（密钥）、密码、私钥。
- 不要默认删除、覆盖、部署、提交、推送。
- 高风险动作先问用户。

## 白话总结

这份示例适合复制到项目根目录后再按项目实际情况调整。
