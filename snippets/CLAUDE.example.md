# 可复制到 CLAUDE.md 的精简规则

默认用中文回答；英文专有名词后加中文说明，例如 skill（技能）、loop（循环）、memory（记忆）、wiki（维基）。

个人 AI 操作系统路径：

- `~/Desktop/myskill`：skill（技能）入口层。
- `~/Desktop/myloop`：loop（循环）工作流层。
- `~/Desktop/my知识库`：Markdown（文档）/ wiki（维基）经验库。

Claude Code 工作规则：

- 简单问题直接回答。
- 复杂任务先判断是否应该启动 `myloop`。
- 可复用经验优先沉淀进 `my知识库`。
- 修改 `myskill`、`myloop`、`my知识库` 前，先读对应目录规则。
- 不要写入 token（令牌）、key（密钥）、密码、私钥、真实账号、机器地址或未脱敏内部信息。
- 每次回答最后加“白话总结”。
