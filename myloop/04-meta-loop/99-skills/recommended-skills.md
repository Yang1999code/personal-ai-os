# 元loop可参考的 skill

这个目录不是执行入口，而是给元loop参考“有哪些 skill 能辅助学习人的处理方式”。

## 核心参考

| skill | 状态 | 用途 |
| --- | --- | --- |
| `my知识库` | 已有，本地 skill | 查历史经验、以前确认过的偏好、可复用规则 |
| `internal-knowledge-skill` | 已有，本地 skill | 当完整任务里涉及内部资料时，补证据 |
| `openai-docs` | 系统内置 | 遇到 OpenAI / Codex / API 相关任务时查官方资料 |
| `skill-check` | 已有，本地 skill | 检查某次任务里提到的 skill 入口是否真的装好 |

## 这些 skill 在元loop中的角色

- 它们不是元loop本身。
- 它们只是帮助元loop回放证据、补充事实、验证环境。
- 元loop真正学习的是“人”，不是 skill。

## 白话总结

这个目录像参考书架。元loop可以借这些 skill 查事实，但不能把“会用 skill”误当成“已经学会这个人”。
