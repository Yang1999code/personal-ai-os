# Engines

## 推荐选择

| 引擎 | 路径 | 适合场景 |
| --- | --- | --- |
| loop1 | `../ai-loop-mini` | 目标偏探索，希望 AI 更自由 |
| loop2 | `../ai-loop-mini2` | 默认推荐，人引导深度循环 |
| loop3 | `../ai-loop-mini3` | 需要 while 脚本、多终端、文件交换 |

## 默认

没有特殊要求时使用：

```text
$HOME/Desktop/myloop/ai-loop-mini2/loop.md
```

这也是当前 `myloop` skill（技能）的稳定入口。
