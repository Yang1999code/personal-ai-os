# Loops循环导航

## 这个入口放什么

这里放 myloop（循环工作流）相关资料。loop 适合复杂任务、长期任务、需要反复对齐和复盘的任务。

## 快速入口

| 页面或目录 | 用途 |
| --- | --- |
| `$HOME/Desktop/myloop` | loop 工作流总目录 |
| `wiki/codex/index.md` | loop 相关经验页索引 |

## 什么时候看这里

- 任务不是一次问答能解决。
- 需要多轮规划、执行、检查、复盘。
- 想把人的处理方式提炼成可复用流程。

## 白话总结

loop 是“复杂任务的跑法”。简单问题直接问 AI，复杂任务才进 loop。
