# Skills技能导航

## 这个入口放什么

这里放 skill（技能）相关资料：已有技能怎么用、什么时候该新建技能、哪些经验适合从 wiki（维基）升级成 skill。

## 快速入口

| 入口 | 用途 |
| --- | --- |
| `$HOME/Desktop/myskill` | 本机 skill 入口层 |
| `$HOME/Desktop/myskill/myloop/SKILL.md` | myloop 启动技能规则 |
| `$HOME/Desktop/myskill/my知识库/SKILL.md` | my知识库写入技能规则 |

## 什么时候该沉淀为 skill

- 任务会重复很多次。
- 有固定输入、固定步骤、固定输出。
- 单靠 wiki 看说明太慢，想让 Agent 自动按流程执行。

## 白话总结

skill 是“可执行的经验”。wiki 先把经验说清楚，重复多了再升级成 skill。
