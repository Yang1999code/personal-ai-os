# 导航入口

`导航/` 是给人和 Agent（智能体）看的前台地图。它只放入口、分类和简短说明，不放大段原始资料。

## 推荐导航页

| 页面 | 用途 |
| --- | --- |
| `AI操作系统.md` | skill、loop、wiki 三层体系 |
| `Loops循环.md` | 复杂任务和 loop 工作流 |
| `Skills技能.md` | skill 入口、触发和升级判断 |
| `工具与排障.md` | 工具链、连接、错误排查 |
| `日常环境.md` | Mac、终端、快捷键等日常经验 |

## 白话总结

导航页就是路牌。它不保存原始资料，只告诉你该去哪里看。
